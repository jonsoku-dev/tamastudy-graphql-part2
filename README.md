# Project 실행 순서

## 레포지토리 클론

```bash
git clone https://github.com/jonsoku2/tamastudy-graphql-part2
```

## 폴더로 이동

```bash
cd tamastudy-graphql-part2
```

## node_module 설치 (root)

```bash
yarn
or
npm install
```

## cli로 프로젝트 설정하기

```bash
yarn cli
or
npm run cli
```
